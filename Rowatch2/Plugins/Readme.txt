Pleasy copy your plugins into this folder.

You can find an example project in the file PluginSample.zip.
Also you can rename the file PluginSample.dl_ to PluginSample.dll to activate the sample plugin.

---

To create a plugin, you can create a new class library project.
Add a reference to RoWatch.Plugin.dll and create a public class inherit from IPlugin.

Compile the dll and copy them to this folder.

Thats all :)
